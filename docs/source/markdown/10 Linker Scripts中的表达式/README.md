##### 4、内建函数

###### 1) ABSOLUTE
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;PS: 节内的地址为相对该节起始的地址。
```
SECTIONS {
    . = 0x100;
    .text : {
        *(.text)
        . = 0x1000;
        *(.init)             /*该部分的VMA为0x1100*/
    }
}
```

> ABSOLUTE

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;ABSOLUTE把节内相对地址转化为绝对地址。

```
SECTIONS {
    . = 0x100;
    .text : {
        *(.text)
        init_begin = ABSOLUTE(0x1000);
        *(.init)             /*该部分的VMA为0x1100*/
    }
}
```
